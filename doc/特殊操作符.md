# Scala特殊操作符
## 数学操作符
- `+`加法
- `-`减法
- `*`乘法
- `/`除法
- `%` 取模
## 位运算操作符
- `|` 位或
- `^` 位异或
- `&` 位与
- `<<` 左移
- `>>` 右移
- `~` 位取反
## 关系运算符
- `==`等于
- `!=`不等于
- `>`大于
- `<`小于
- `>=`大于等于
- `<=` 小于等于
## 逻辑运算符
- `&&` 与
- `||` 或
- `!` 非

## to
- 返回一个`Range`对象
```scala
val inclusive = 1 to 10
val inclusive1 = 1.to(10)
```
